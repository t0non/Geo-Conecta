# Geo-Conecta | Consultoria Geológica e Ambiental

Este repositório contém o código-fonte da plataforma Geo-Conecta, uma consultoria boutique especializada em projetos de alta complexidade em geociências e engenharia mineral.

## Funcionalidades e Otimizações Recentes
- **Arquitetura de Autoridade Técnica**: Inclusão de checklists de entregáveis (`EscopoTecnico.tsx`) e cards de risco regulatório (`SinaisAlerta.tsx`).
- **Sistema de Prova Social**: Integração de depoimentos animados (`Authority.tsx`) para reforçar a credibilidade técnica.
- **UX Consolidada**: Centralização da conectividade via WhatsApp em um componente global único, eliminando redundâncias visuais.
- **Design System Premium**: Interface minimalista com foco em conversão B2B, utilizando Tailwind CSS e Framer Motion.

## Como Rodar Localmente

**Pré-requisitos:** Node.js (v18+)

1.  **Instale as dependências:**
    `npm install`
2.  **Configure as variáveis de ambiente:**
    Defina a `GEMINI_API_KEY` no arquivo `.env.local`.
3.  **Inicie o servidor de desenvolvimento:**
    `npm run dev`
